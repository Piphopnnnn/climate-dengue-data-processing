# Climate and Dengue Data Processing

This project focuses on preprocessing and integrating climate and dengue-related datasets across Thailand for statistical and forecasting research.

---

## Project Objectives

- Process nationwide climate and epidemiological datasets
- Clean and validate multi-source weather data
- Aggregate time-series datasets into structured formats
- Prepare datasets for downstream statistical and forecasting analysis

---

## Technologies and Tools

- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## Workflow

1. Load climate and dengue datasets
2. Clean and preprocess raw datasets
3. Aggregate weather data into structured time-series formats
4. Merge multi-source datasets
5. Prepare datasets for research and forecasting workflows

---

## Project Files

- `combine_data.ipynb` : Combine and merge datasets
- `DF_clean.ipynb` : Dengue fever data preprocessing
- `DHF_clean.ipynb` : Dengue hemorrhagic fever preprocessing
- `rain_new.ipynb` : Rainfall preprocessing and aggregation

---

## Applications

- Climate data analysis
- Dengue-related research
- Time-series data preparation
- Forecasting workflow preparation

---

## Notes

This repository contains preprocessing and data integration workflows only.  
Raw datasets and research-related data are not publicly included.
